Pacing Guide

Implementation options and schedules for the Metanoia curriculum

The Metanoia curriculum is designed to be flexible and adaptable to different school schedules and timeframes. This guide presents three implementation options depending on your specific context, with detailed breakdowns of each approach.

Standard (8 Weeks) Intensive (2 Weeks) Extended (16 Weeks) Adaptations

Standard 8-Week Implementation

Format: 2 sessions per week, 90 minutes each

Implementation Adaptations

The Metanoia curriculum is designed to be flexible and adaptable to different contexts. Here are suggestions for common adaptation scenarios:

Limited Time

If you have less than 8 weeks available:

  • Focus on Modules 1, 2, 5, and 8 as a core experience
  • Combine related modules (e.g., Modules 3-4 or Modules 6-7)
  • Assign some reflective activities as homework
  • Create a "taster" curriculum using one activity from each module

Limited Resources

For schools with minimal material resources:

  • Use the alternative materials suggestions in each module
  • Focus on discussion-based and observational activities
  • Substitute digital tools with analog alternatives
  • Emphasize community walks and local observations
  • Encourage reuse of available waste materials for activities

Younger Students

For adapting to junior secondary or primary levels:

  • Simplify conceptual frameworks (e.g., basic version of Iceberg Model)
  • Increase time for hands-on activities
  • Use more visual aids and concrete examples
  • Focus on immediate, visible impacts in the local environment
  • Incorporate more games and role-playing activities

Cross-Curricular Integration

For integrating across multiple subject areas:

  • Module 1: Science/Geography
  • Module 2: Economics/Business Studies
  • Module 3: Mathematics/Social Studies
  • Module 4: Art/Design
  • Module 5: Technology/Engineering
  • Module 6: Civics/Social Studies
  • Module 7: Language Arts/Media Studies
  • Module 8: Life Skills/Career Guidance

Large Class Sizes

For classes with 40+ students:

  • Use a "station" approach with rotating small groups
  • Implement a peer leadership system with student facilitators
  • Create larger collaborative groups (6-8 students)
  • Use gallery walks for sharing instead of presentations
  • Adapt mapping activities to wall-sized group work

Exam-Focused Contexts

For schools with strong examination focus:

  • Explicitly connect activities to curriculum standards
  • Include more structured assessment opportunities
  • Build in skills practice for analytical writing and problem-solving
  • Create links to examination question formats
  • Provide sample questions that apply curriculum concepts

Implementation Notes:

  • Each timeline can be adjusted based on specific school schedules and student needs
  • Additional time may be needed for field trips, guest speakers, or extended projects
  • Assessment activities can be integrated into sessions or assigned as homework
  • Larger schools may consider a staggered implementation across multiple classes
  • Consider curriculum connections when scheduling to align with related subjects
